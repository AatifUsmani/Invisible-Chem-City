# Quickstart Guide

## 1. Prerequisites
- **Mapbox Token:** Get a free public token from [mapbox.com](https://mapbox.com).
- **Python 3.9+** & **Node.js 18+**.

## 2. Fast Track (Run Web App)
The project comes with a pre-processed `facilities.json`. You can see the site immediately:
```bash
cd web
npm install
# Add VITE_MAPBOX_TOKEN to .env
npm run dev
